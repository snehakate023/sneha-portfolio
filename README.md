How to host (quick options):
A) GitHub Pages
 - Create a new GitHub repository.
 - Push all files to the repo.
 - In repository settings > Pages, set branch to main (or gh-pages) and root folder '/'.
 - Your site will be available at https://<username>.github.io/<repo-name>

B) Netlify (drag & drop)
 - Create a Netlify account.
 - Drag and drop the zip or the site folder to deploy.
 - Set site name if desired.

C) Vercel
 - Create an account and import the repo. Vercel will deploy automatically.
